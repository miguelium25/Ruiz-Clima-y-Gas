# Ruiz Clima y Gas

Landing page moderna y optimizada para **Instalaciones Ruiz Clima y Gas**, especialistas en calderas, aire acondicionado y calefacción en Barcelona.

## Características

- **Diseño Responsivo:** Adaptado a móviles, tablets y escritorio.
- **Optimización SEO Local:** Estructura semántica y metadatos para posicionamiento en Barcelona.
- **Llamadas a la Acción (CTA):** Botones flotantes de WhatsApp y teléfono.
- **Formularios de Contacto:** Validación integrada.
- **Marcas Reales:** Logos SVG de los principales fabricantes.

## Tecnologías

- HTML5 Semántico
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Vanilla, ES6+)

## Instalación

Para ver el proyecto localmente:

1. Clonar el repositorio (cuando esté disponible).
2. Abrir `index.html` en tu navegador.


## Despliegue (Hosting Gratuito)

Este proyecto está listo para ser alojado en **GitHub Pages**.

1. Ve a la pestaña **Settings** (Configuración) de este repositorio.
2. En el menú lateral izquierdo, haz clic en **Pages**.
3. En **Build and deployment** > **Source**, selecciona `Deploy from a branch`.
4. En **Branch**, selecciona `main` y carpeta `/ (root)`.
5. Haz clic en **Save**.

En unos minutos, tu web estará online en:  
`https://miguelium25.github.io/Ruiz-Clima-y-Gas/`
